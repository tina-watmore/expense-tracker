Core purpose: 

- Set monthly budget limits for different categories 
- Track daily expenses and show remaining monthly budget for categories 
- Roll over any remainder (plus or negative)


- Add / edit / delete expenses
- view totals / graphs
- use database 

Cards: 
 - Networth section
	- assets
	- properties 
	- debt
	- savings
	- superannuation
	- investments
 - section for properties 	
	- rent
	- expenses
	- holding cost
